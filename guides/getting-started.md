# From Scratch

- install homebrew
- install xcode (from itunes store: http://itunes.apple.com/us/app/xcode/id497799835?mt=12)
  - click "view in the mac app store" (blue button on left)
  - you may have to update your OS
- brew install git
- brew install node
- brew install mongodb
- brew install redis
